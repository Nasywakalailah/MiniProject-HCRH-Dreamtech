A Technology, Innovation, Research & Development Company. Infinite Learning, one of the divisions of PT Kinema Systrans multimedia (a subsidiary of Infinite Studios), centres on developing vocational training courses relevant to Infinite Studios' activities and the growing demand for skilled talent in the Nongsa Digital Park ecosystem.

Infinite Learning's Programs

  • MSIB Mandiri: A Certified Internship and Independent Study Program.
  • Magang Flagship: A Flagship Internship Program.
  • Magang Mandiri: An Independent Internship Program.
  • Bootcamp: A Bootcamp Program.

Hybrid Cloud & Red Hat

The Hybrid Cloud and Red Hat program prepares participants to become Linux system administrators. They will learn installation, networking, storage, security, and advanced topics such as LVM, SELinux, automation, data security, and cloud computing with hands-on practice using IBM Cloud and Red Hat Openshift.

Monitor Process in Linux

Poster dengan penjelasan terkait proses dalam linux dan cara memonitorinya.

Poster ini memberikan gambaran terkait monitoring proses di Linux, dengan fokus pada process states, process types, dan bagaimana proses-proses tersebut dapat dikelola. Sebuah proses di Linux sendiri adalah instansi yang berjalan dari sebuah program, yang diberi tanda pengenal unik yang disebut Process ID (PID).

  • Client: Infinite Learning
  • Category: Informative Poster

Real-time Monitoring with top&htop

Poster dengan penjelasan terkait command 'top' dan 'htop' dalam konteks process monitoring.

Poster ini menjelaskan fungsi dan perbedaan dari command top dan htop secara umum. Pada poster juga dijelaskan opsi-opsi dalam penggunaan top dan htop.

  • Client: Infinite Learning
  • Category: Informative Poster

Ps Command Review

Poster dengan penjelasan terkait command 'ps' dalam konteks process monitoring.

Poster ini menggambarkan dan menjelaskan tentang bagian-bagian (columns) yang akan muncul saat menggunakan command ps. Ada juga opsi-opsi yang umum digunakan bersamaan dengan command ps.

  • Client: Infinite Learning
  • Category: Informative Poster

Summary and Best Practices in Process Management

Poster dengan penjelasan terkait kesimpulan dan hal-hal yang baik dilakukan dalam process monitoring.

Poster ini menjelaskan ulang secara singkat tentang hal-hal yang dibahas pada poster-poster yang lain terkait GUI Tools dan command-command seperti ps, top, htop, dan grep. Ada juga tips yang diberikan seperti selalu gunakan command kill seperlunya dan jangan langsung paksa berhenti suatu proses.

  • Client: Infinite Learning
  • Category: Informative Poster

Grep, Pipeline, and Redirect/Append in Process Monitoring

Poster dengan penjelasan terkait command 'grep' beserta konsep pipeline dan redirect/append dalam konteks process monitoring.

Poster ini menggambarkan dan menjelaskan secara singkat tentang penggunaan command grep beserta options yang biasa digunakan dalam process monitoring. Poster juga menjelaskan sedikit tentang konsep pipeline dan redirect/append beserta penggunaannya dalam process monitoring.

  • Client: Infinite Learning
  • Category: Informative Poster

Common Task Manager for Linux

Poster dengan penjelasan terkait GUI-based task manager dan kelebihannya dalam konteks process monitoring.

Poster ini menggambarkan berbagai GUI-based task manager yang ada pada berbagai distro Linux dengan Desktop Environment yang juga berbeda-beda. Ada juga sedikit penjelasan terkait kelebihan dari GUI tools seperti task manager ini dalam process monitoring.

  • Client: Infinite Learning
  • Category: Informative Poster
